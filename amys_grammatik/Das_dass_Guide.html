DAS / DASS

NMS 1. Klasse Nachprüfung - Kompletter Leitfaden für die Note 1

🎯 Grundlagen verstehen

DIE GOLDENE REGEL:

DAS kann man durch DIESES, JENES oder WELCHES ersetzen.

DASS kann man NIEMALS ersetzen - es leitet einen Nebensatz ein.

📝 DAS (mit einem S)

  • Artikel: das Haus, das Auto
  • Demonstrativpronomen: Das ist schön
  • Relativpronomen: Das Buch, das ich lese
Test: Das Auto → Dieses Auto ✓

📝 DASS (mit zwei S)

  • Konjunktion: Verbindet Sätze
  • Leitet Nebensätze ein
  • Nicht ersetzbar!
Test: Ich weiß, dass... → Ich weiß, dieses... ✗

🔍 Der ultimative ERSETZUNGSTEST

🎯 SO FUNKTIONIERT DER TEST:

1
Setze DIESES, JENES oder WELCHES ein

2
Macht der Satz noch Sinn?

3
JA = DAS (ein S) | NEIN = DASS (zwei S)

Beispiele mit Test:

Beispiel 1: "Das Buch liegt auf dem Tisch."
Test: "Dieses Buch liegt auf dem Tisch." ✓
→ DAS (ein S)
Beispiel 2: "Ich glaube, dass es regnet."
Test: "Ich glaube, dieses es regnet." ✗
→ DASS (zwei S)
Beispiel 3: "Das, was du sagst, ist richtig."
Test: "Dieses, was du sagst, ist richtig." ✓
→ DAS (ein S)

📝 DAS (mit einem S) - Alle Verwendungen

1. DAS als bestimmter Artikel

Verwendung: Vor Substantiven im Neutrum (sächlich)

Erkennbar an: Steht direkt vor einem Nomen

Beispiel Ersetzungstest Erklärung
Das Kind spielt. Dieses Kind spielt. ✓ Artikel vor "Kind"
Das Wetter ist schön. Dieses Wetter ist schön. ✓ Artikel vor "Wetter"
Das Haus ist groß. Dieses Haus ist groß. ✓ Artikel vor "Haus"
2. DAS als Demonstrativpronomen

Verwendung: Zeigt auf etwas hin ("das da", "das hier")

Erkennbar an: Steht allein, oft am Satzanfang

Beispiel Ersetzungstest Erklärung
Das ist mein Freund. Dieses ist mein Freund. ✓ Zeigt auf eine Person
Das gefällt mir nicht. Dieses gefällt mir nicht. ✓ Zeigt auf eine Sache
Das kann nicht wahr sein. Dieses kann nicht wahr sein. ✓ Zeigt auf eine Aussage
3. DAS als Relativpronomen

Verwendung: Leitet Relativsätze ein (Nebensätze mit Zusatzinfo)

Erkennbar an: Folgt nach einem Komma, bezieht sich auf ein vorheriges Wort

Beispiel Ersetzungstest Erklärung
Das Auto, das dort steht, ist neu. Das Auto, welches dort steht, ist neu. ✓ Relativsatz zu "Auto"
Das Buch, das ich lese, ist spannend. Das Buch, welches ich lese, ist spannend. ✓ Relativsatz zu "Buch"
Das Kind, das weint, braucht Hilfe. Das Kind, welches weint, braucht Hilfe. ✓ Relativsatz zu "Kind"

📝 DASS (mit zwei S) - Die Konjunktion

DASS ist immer eine Konjunktion!

Es verbindet einen Hauptsatz mit einem Nebensatz und kann NIEMALS ersetzt werden!

Typische Wendungen mit DASS

🗣️ Verben des Sagens/Denkens

  • Ich sage, dass ...
  • Er meint, dass ...
  • Sie denkt, dass ...
  • Wir glauben, dass ...
  • Sie behauptet, dass ...
Beispiel: Ich weiß, dass du recht hast.
Test: Ich weiß, dieses du recht hast. ✗

💭 Verben des Fühlens/Empfindens

  • Ich hoffe, dass ...
  • Er fürchtet, dass ...
  • Sie freut sich, dass ...
  • Wir befürchten, dass ...
  • Ich ärgere mich, dass ...
Beispiel: Sie hofft, dass alles gut wird.
Test: Sie hofft, welches alles gut wird. ✗

👁️ Verben des Wahrnehmens

  • Ich sehe, dass ...
  • Er bemerkt, dass ...
  • Sie stellt fest, dass ...
  • Wir erkennen, dass ...
  • Man merkt, dass ...
Beispiel: Ich merke, dass du müde bist.
Test: Ich merke, jenes du müde bist. ✗

💡 Unpersönliche Ausdrücke

  • Es ist wichtig, dass ...
  • Es ist schön, dass ...
  • Es ist schade, dass ...
  • Es tut mir leid, dass ...
  • Es ist klar, dass ...
Beispiel: Es ist schön, dass ihr da seid.
Test: Es ist schön, dieses ihr da seid. ✗

⚠️ Schwierige Fälle und Fallen

1. "Das, was..." - Konstruktionen

Regel: Bei "das, was..." steht immer DAS (ein S)

Grund: "Das" ist hier ein Demonstrativpronomen

Richtig Falsch Ersetzungstest
Das, was du sagst, ist richtig. Dass, was du sagst, ist richtig. Jenes, was du sagst, ist richtig. ✓
Das, was passiert ist, tut mir leid. Dass, was passiert ist, tut mir leid. Dieses, was passiert ist, tut mir leid. ✓
Das, was ich meine, ist folgendes. Dass, was ich meine, ist folgendes. Jenes, was ich meine, ist folgendes. ✓
2. Satzanfang-Situationen

Am Satzanfang: Meist DAS (ein S) als Demonstrativpronomen

DASS am Satzanfang ist sehr selten!

Beispiel Erklärung Ersetzungstest
Das kann nicht sein! Demonstrativpronomen Dieses kann nicht sein! ✓
Das ist mein Ernst. Demonstrativpronomen Jenes ist mein Ernst. ✓
Das macht keinen Sinn. Demonstrativpronomen Welches macht keinen Sinn. ✓
3. Nach Komma - Die große Verwechslungsgefahr

Nach Komma kann beides stehen!

Entscheidend: Der Ersetzungstest!

Vergleiche diese Sätze:

DAS (Relativpronomen):
"Das Buch, das dort liegt, gehört mir."
Test: "Das Buch, welches dort liegt, gehört mir." ✓
→ DAS (ein S)
DASS (Konjunktion):
"Ich weiß, dass das Buch dort liegt."
Test: "Ich weiß, dieses das Buch dort liegt." ✗
→ DASS (zwei S)
4. Doppelte das/dass-Konstruktionen

🎯 Wenn beide vorkommen:

Manchmal stehen DAS und DASS im selben Satz!

Regel: Jeden Fall einzeln mit dem Ersetzungstest prüfen!

Beispiel Analyse Erklärung
Ich weiß, dass das Auto kaputt ist. dass = Konjunktion
das = Artikel		 dass: leitet Nebensatz ein
das: Artikel vor "Auto"
Er sagt, dass das, was passiert ist, schlimm war. dass = Konjunktion
das = Demonstrativpronomen		 dass: leitet Nebensatz ein
das: "jenes, was passiert ist"

🧠 Merkhilfen und Eselsbrücken

🎯 Die ultimative Eselsbrücke

"DAS kann man durch DIESES ersetzen!"

Beide fangen mit D an und haben nur ein S!

🔄 Der Austausch-Trick:

  • DAS → DIESES/JENES/WELCHES (funktioniert!)
  • DASS → DIESES/JENES/WELCHES (funktioniert NICHT!)

📚 Merkspruch für DASS:

"Ich sage, denke, fühle, dass..."

Nach Verben des Sagens, Denkens und Fühlens steht meist DASS!

✅ Wann DAS (ein S)?

  • Vor Substantiven (das Haus)
  • Am Satzanfang (Das ist toll!)
  • Bei "das, was..." Konstruktionen
  • Nach Komma + Ersetzung möglich

✅ Wann DASS (zwei S)?

  • Nach "ich sage/denke/hoffe..."
  • Nach "es ist wichtig/schön..."
  • Keine Ersetzung möglich!
  • Verbindet Haupt- und Nebensatz

✏️ Übungen für Profis

🎯 Übung 1: Grundlagen-Check

Entscheide: DAS oder DASS?

1. Ich glaube, ____ du recht hast.

2. ____ Buch gehört mir.

3. Es tut mir leid, ____ ich zu spät bin.

4. ____ ist eine gute Idee!

5. ____ Auto, ____ dort steht, ist neu.

📝 Lösungen:

1. DASS (leitet Nebensatz ein) | 2. DAS (Artikel) | 3. DASS (nach "es tut mir leid") | 4. DAS (Demonstrativpronomen) | 5. DAS, DAS (Artikel + Relativpronomen)

🎯 Übung 2: Ersetzungstest anwenden

Prüfe mit DIESES/JENES/WELCHES:

1. Ich hoffe, ____ alles gut wird.

2. ____, was du sagst, stimmt.

3. Er meint, ____ wir früher gehen sollten.

4. ____ Problem ist schwierig.

📝 Mit Ersetzungstest:

1. "Ich hoffe, dieses alles gut wird." ✗ → DASS

2. "Jenes, was du sagst, stimmt." ✓ → DAS

3. "Er meint, welches wir früher gehen sollten." ✗ → DASS

4. "Dieses Problem ist schwierig." ✓ → DAS

🎯 Übung 3: Schwierige Fälle

Die Königsdisziplin:

1. ____, was passiert ist, zeigt, ____ wir vorsichtiger sein müssen.

2. Ich sehe, ____ ____ Kind, ____ dort spielt, glücklich ist.

3. Es ist wichtig, ____ ____ Thema, ____ wir besprechen, klar ist.

📝 Schritt-für-Schritt Analyse:

1. DAS (Demonstrativ), DASS (nach "zeigt")

2. DASS (nach "ich sehe"), DAS (Artikel), DAS (Relativpronomen)

3. DASS (nach "es ist wichtig"), DAS (Artikel), DAS (Relativpronomen)

🏆 Tipps für die Nachprüfung

⭐ Die 3 goldenen Regeln für die Note 1:

  1. Immer den Ersetzungstest machen! Kann ich DIESES/JENES/WELCHES einsetzen?
  2. Nach Verben des Sagens/Denkens/Fühlens steht meist DASS
  3. Bei "das, was..." steht immer DAS (ein S)

✅ Prüfungsstrategie

  • Text erst komplett lesen
  • Jeden das/dass-Fall einzeln prüfen
  • Ersetzungstest bei Unsicherheit
  • Kontrolle am Ende

⚡ Schnell-Check

  • Nach Komma + Verb → meist DASS
  • Vor Substantiv → meist DAS
  • Am Satzanfang → meist DAS
  • Nach "ich sage/denke..." → DASS

🎯 Häufige Fallen

  • "Das, was..." → immer DAS!
  • Doppelte Konstruktionen einzeln prüfen
  • Nach Komma nicht automatisch DASS
  • Ersetzungstest nie vergessen

🔄 Letzte Kontrolle

  • Jeden Fall nochmals überprüfen
  • Ersetzungstest bei Zweifeln
  • Typische Wendungen kontrollieren
  • Ruhe bewahren und systematisch vorgehen

🎊 Zusammenfassung - Alles auf einen Blick

🏆 DER ULTIMATIVE SPICKZETTEL

DAS (ein S): Ersetzbar durch DIESES/JENES/WELCHES

DASS (zwei S): NICHT ersetzbar - leitet Nebensatz ein

Situation Regel Beispiel Test
Vor Substantiv DAS (Artikel) Das Haus ist groß. Dieses Haus ist groß. ✓
Am Satzanfang Meist DAS (Demonstrativ) Das ist richtig! Jenes ist richtig! ✓
Nach Komma + Substantiv DAS (Relativpronomen) Das Buch, das ich lese... Das Buch, welches ich lese... ✓
"das, was..." Immer DAS Das, was du sagst... Dieses, was du sagst... ✓
Nach Sagen/Denken/Fühlen DASS (Konjunktion) Ich glaube, dass... Ich glaube, dieses... ✗
"Es ist... dass" DASS (Konjunktion) Es ist schön, dass... Es ist schön, welches... ✗
